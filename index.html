§ 02 · The Caravan Loop

Open. Load. Dispatch. Travel. Arrive.

Five phases. A heartbeat every 20 game ticks. A snapshot every 100. A server restart never loses a caravan in flight.

i.
Open
/caravan trade opens a 54-slot dispatch GUI. Slot 49 shows the live destination value of what you've loaded so far.
/caravan trade
ii.
Load
Slots 0–44 hold real ItemStacks — NBT, durability, enchants and CMD preserved. Items serialise to Base64; the llama then spawns.
Trader llama · 45 cargo slots
iii.
Dispatch
Min cargo enforced (min-cargo-value). Source land's supply drains 10% of shipped quantity. Markers post to every enabled MapHook.
Squaremap · Dynmap · BlueMap
iv.
Travel
Waypoint pathfinding, 20-block step cap, 4-block arrival check. Anti-teleport at 50 blocks/tick. Wilderness flag is one-way — once vulnerable, stays vulnerable.
teleport-threshold = 50
v.
Arrive
Destination land's supply replenishes. Net payout = base − tax − maintenance. Stocks tick. Dividends accrue. The ledger writes a line.
net = base − tax − maint.

Twelve systems. One economy.

Each module is opt-in. Run only the caravan loop, or wire the whole engine: four-type exchange, dividends, three-tier guards, embargoes, contracts, built-in map renderer, region protection, and Fog of War.

/ 01
Per-land price engine
price = clamp(base × (BASE_SUPPLY/supply)^sensitivity, min, max). Source drains on dispatch, destination replenishes on arrival, hourly recovery nudges every cache back toward baseline.

/ 02
Four-type stock exchange
Route stocks (+5% / −15%), Land stocks (activity score), Commodity indices (server-wide S/D), and Company shares — all on the same order book, with OHLC history.
/caravan stocks · chest GUI + REST
/ 03
Four company types
Every player owns one PLAYER company. Lands and nations may list LAND and NATION shares. Admins charter ROUTE companies on specific lanes.

Trade agreements flow PENDING → ACTIVE → EMBARGO. Import tax up to 15% paid to the land owner. Embargoes block direct dispatch entirely.

Three-tier guard escorts
Vindicator NPCs at Tier I / II / III against PvP and bandit raids. Bandit camps spawn fully-configurable ambushes — Trade Crate drops and vault-paid bounties.

§ 09 · Things people ask

Plainly answered.

Six questions every server admin sends before clicking buy. Honest answers below.

i. Does it work alongside my existing economy plugin? +

Yes. ScappiTrade routes every gold movement through Vault, so any Vault-compatible economy (EssentialsX, CMI, TheNewEconomy, etc.) is the single source of truth for balances. ScappiTrade only persists holdings, orders, and ledger — not balances.

ii. What if I don't use Lands or Towny? +

You can still run the entire caravan loop and exchange. Without a land plugin, tax windows fall back to per-world brackets defined in tax.yml. Lands/Towny is recommended (the diplomacy layer needs jurisdictions to be interesting) but not required.

iii. Will dispatching caravans tank my server's TPS? +

The heartbeat ticks every 20 game ticks (1 second) with a 20-block pathfinding step cap. Real-world: a server running 60 simultaneous caravans on a Ryzen 5 5600 sits around 0.4ms / tick. The map proxy and exchange run on async threads, not the main thread.

iv. Do I need to expose a port for the web dashboard? +

The dashboard binds to 0.0.0.0:25578 by default and is fully optional. For production, put Nginx or Caddy in front for HTTPS — the docs cover this explicitly. If you don't want a dashboard at all, leave the analytics module disabled in config.yml.

v. Can I customise prices, taxes, guard tiers? +

Yes. config.yml has 550+ keys covering supply sensitivity, recovery rates, tax brackets, guard tiers, ambush probabilities, dividend cadence, share-crash percentages, and ledger retention. The item catalogue is a folder of YAML files — add or edit categories without restarting.

vi. What's the license really say? +

One Minecraft server per license, any player count. Lifetime updates including the 2.x line. No telemetry. No license server phone-home. The JAR is IP-locked at first start. Network (BungeeCord/Velocity) licenses on request — ask before you buy.
